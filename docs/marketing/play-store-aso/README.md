# Dosifi Play Store ASO Pack (v1)

This folder contains 3 complete Google Play listing variants for store experiments.

## Files
- `variant-a-balanced.md` — balanced broad terms (medicine tracking + scheduling)
- `variant-b-vials-peptides.md` — niche intent (reconstitution + vials + peptides)
- `variant-c-reminders-adherence.md` — reminder/log intent (dose log + adherence + reminders)

## Suggested Test Order
1. Run Variant A as baseline (2-3 weeks)
2. Test Variant B against A (title + short description first)
3. Test Variant C against winner (title + short description)
4. Keep the best pair and then test full-description changes

## Success Metrics
- Store listing conversion rate (visitors to installers)
- First-time installers
- 7-day retention
- Review sentiment mentioning key use cases

## Notes
- Google Play does not use a hidden keyword field. Keyword relevance comes from title, short description, full description, reviews, and engagement signals.
- Keep language natural. Avoid stuffing repeated keywords.
